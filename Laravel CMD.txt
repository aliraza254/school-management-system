Start Creating With Folder:
composer create-project laravel/laravel example-app

Install Node Modules
npm install

Install Modules Globally
npm install -g node-modules

Run Laravel Server:
php artisan serve

To Creat All Files Mode/Factory/Migration/Seeder/Request/Request/Controller/Policy:
php artisan make:model Teacher -a

To Creat Model/Controller/Migration Files:
php artisan make:model Customer -mcr

Refresh Data Base:
php artisan migrate:refresh

Refresh Specific Data Base
php artisan migrate:refresh --path="database/migrations/2023_01_21_180849_create_teachers_table.php"

Clear Project Cache:
php artisan cache:clear


php artisan key:generate

Check Route List
php artisan route:list --name=student
